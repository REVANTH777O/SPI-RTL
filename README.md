## Overview

This repository implements a complete Serial Peripheral Interface (SPI) communication system using synthesizable Verilog RTL. The project consists of an SPI Master (Transmitter), SPI Slave (Receiver), Top Module, and Verification Testbench developed in Xilinx Vivado.

The design follows a modular RTL approach and demonstrates the complete SPI communication flow from data transmission to reception.

This project is intended for students, FPGA developers, RTL Design Engineers, and interview preparation for Digital Design roles.

# SPI Protocol Overview

SPI (Serial Peripheral Interface) is a synchronous full-duplex communication protocol used for high-speed communication between a Master device and one or more Slave devices.

Unlike UART, SPI uses a clock generated by the Master.

Signals:

- MOSI – Master Out Slave In
- MISO – Master In Slave Out
- SCLK – Serial Clock
- CS/SS – Chip Select
